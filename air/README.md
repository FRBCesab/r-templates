# Instructions

These files must be stored as follow:

```
project/
├── .vscode/
│   ├── extensions.json
│   └── settings.json
└── air.toml
```

You need to install the [air extension](https://posit-dev.github.io/air/editor-vscode.html#installation)  for Positron IDE or the [air cli](https://posit-dev.github.io/air/editor-rstudio.html#installation) for other IDE (e.g. RStudio).
